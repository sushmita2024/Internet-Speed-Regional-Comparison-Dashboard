📌 Project Overview

The Internet Speed Regional Comparison Dashboard is a web-based application that allows users to analyze and compare internet speeds across different regions. It provides insights into download speed, upload speed, latency, and overall network performance. This project helps users, researchers, and organizations understand internet quality trends and make informed decisions.
